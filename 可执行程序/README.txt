使用"启动命令"+"参数"可调用此程序功能
Start "" "Main.exe" -type=0/1/2/3/4 -title="<String>" -text="<Text>"

-type:
0 = 无图标
1 = 提示(蓝色圆形感叹号)
2 = 疑问(蓝色圆形问号)
3 = 警告(黄色三角形感叹号)
4 = 错误(红色圆形叉号)

-title:
字符串;用于设置弹窗的标题

-text:
字符串;用于设置弹窗的内容

---

Q: 所有参数都必填吗?不填某个参数会怎样?
A: 是的所有参数都必填,诺少填参数则不会运行
Q: 参数对大小写字母有要求吗?
A: 没有
Q: 如果重复输入单个参数会怎样?
A: 优先执行最靠后的这个参数
Q: 入口"Main.exe"是否可以更改名称?
A: 可以
Q: 库"MsgBox.dll"是否可以更改名称?
A: 不可以
